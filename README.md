📌 Project Description

This is a Personal Portfolio Website built using HTML, CSS, and JavaScript to showcase skills, projects, education, and contact information.

The portfolio acts as a digital resume and helps present your work professionally to recruiters, companies, and clients.

🎯 Features
🏠 Home section (Introduction & profile)
👨‍💻 About section (skills & background)
📁 Projects section (showcase your work)
📜 Resume download option
📞 Contact section (email / form)
🌙 Responsive design (mobile-friendly)
🎨 Clean and modern UI
🛠️ Technologies Used
HTML5 – Structure of the website
CSS3 – Styling and layout
JavaScript – Interactivity and dynamic behavior
